# 快捷键

Jupyter笔记本有两种不同的键盘输入模式，**编辑模式** 允许将代码或文本输入到单元格，并以绿色单元格边框表示。**命令行模式** 允许输入各种命令，并有具有蓝色左边框的灰色单元格边框表示。

从命令行模式进入编辑模式按`Enter`键，从编辑模式切换到命令行模式按`Esc`键。

下表将对快捷键做简单的说明：

| Ctrl-Home           | 跳到单元开头         |                                          |

| Ctrl-End            | 跳到单元末尾         |                                          |
