### 提升github的逼格

##### github上可以添加表情啦，

- 在github上插入表情的使用方法：
  - 使用方法为前后冒号包围表情代号的句法。````:blush:``` 会显示成对应的表情，所以你在提交代码的时候可以这样写提交信息：`

git commit –m ‘commit some changes :blush:’
\```
